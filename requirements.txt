This project is a Streamlit web application that uses the Gemini AI to analyze a YouTube video's transcript. It determines how relevant the video's content is to a given title and description, providing a "Relevance Score" and other key insights.

üöÄ Core Features

Relevance Scoring: Get an instant "Relevance Score" (0-100%) that compares the video's transcript to your title and description.

AI-Powered Justification: An AI-generated explanation for why the video received its score.

Content Flagging: Automatically flags videos with "Irrelevant Segments" (like sponsorships or off-topic rants).

Key Insights: Automatically extracts the "Top 5 Key Points" and a "Quarterly Summary" (25%, 50%, 75%, 100%) of the video.

Category Tags: The AI suggests relevant tags (e.g., Tech, Education, How-to) for the video's content.

Follow these steps to run the application on your own computer.

1. Clone or Download the Repository

First, get the code from this repository.

git clone [https://github.com/YOUR-USERNAME/ai-video-relevance-analyzer.git](https://github.com/YOUR-USERNAME/ai-video-relevance-analyzer.git)
cd ai-video-relevance-analyzer


2. Install Dependencies

Install all the required Python libraries.

pip install -r requirements.txt


3. Create Your Secret API Key

This app requires a Google Gemini API key.

Create a new folder named .streamlit in the project directory.

Inside that folder, create a new file named secrets.toml.

Paste the following into that file, adding your own API key:

GEMINI_API_KEY = "YOUR_AIza...KEY_HERE"


4. Run the App

Launch the Streamlit app from your terminal.

streamlit run app.py


Your app will open in your browser at http://localhost:8501.
